Description / 描述

This project is undertaken simultaneously as a project for the MSc in Data Science and Machine Learning course "DSML Industrial Consulting and Applications Project". The aim of this project is to build a Proof-of-Concept (POC) face recognition login system for Occutrack's flagship web-based eye monitoring software, which is also augmented by a POC Presentation Attack Detection (PAD) system for further security purposes. The full report for this project can be downloaded from the left panel. This is part of a larger project that involves developing a complete POC web-based eye monitoring software which will be used by patients and clinicians in monitoring for the presence or possible deterioration of eye condition for patients who are currently suffering from (or will suffer from) Age-related Macular Degeneration (AMD). The POC web-based eye monitoring software can be accessed here for demonstration purposes.
